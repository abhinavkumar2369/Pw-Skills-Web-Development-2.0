1. Bold tag
    • This tag is used to make text bold.
  2. Italic Text
    • This tag is used to make text italic.
  3. Underline Text
    • This tag is used to underline text.
  4. Strikethrough Text
    • This tag is used to apply a strikethrough effect to text.
  5. Subscript Text
    • This tag is used to render text as subscript (e.g., chemical formulas)
  6. Superscript Text
    • This tag is used to render text as superscript (e.g., exponent notation).
  7. Inline Code
    • This tag is used to display code or computer instructions within the text.
  8. Preformatted Text
    • This tag is used to display text in a fixed-width font, maintaining spacing and line breaks.
  9. Blockquote
    • This tag is used to create a block of indented text, often used for quotations.
  10. Emphasized Text
    • This tag is used to emphasize text, typically displayed in italics.
