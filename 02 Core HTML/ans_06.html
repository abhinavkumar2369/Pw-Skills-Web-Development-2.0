1. html tag
    • The root element of an HTML document that encapsulates all other HTML elements.
    • Every HTML document must start with this tag.
  2. head tag
    • Contains metadata about the HTML document, such as the title, character encoding, and linked stylesheets.
    • Information placed here is not visible on the webpage but is crucial for various purposes like SEO and browser rendering.
  3. title tag
    • Sets the title of the webpage, which is displayed in the browser's title bar or tab.
    • Helps users identify the page when multiple tabs are open and is also important for search engine optimization.
  4. table tag
    • Defines a table for organizing data into rows and columns.
    • Used to display tabular data, such as schedules, data comparisons, and more.
  5. body tag
    • It defines the body potion of the web page
    • It tell in main content of the webpage which will be displayed
